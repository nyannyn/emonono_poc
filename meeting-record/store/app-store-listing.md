## 描述
Speech-to-Text 是一款會議轉譯工具：

• 錄音或上傳音檔，自動產生繁體中文逐字稿
• AI 整理成會議摘要、行動項目、討論重點、待釐清事項
• 講者分離：自動標出不同說話者
• 成員聲紋註冊，逐字稿自動對應姓名
• 即時字幕模式
• 會議歷史、搜尋、音檔播放與匯出

轉錄與摘要由 OpenAI 技術提供。預設使用內建額度，免設定即可開始；進階使用者也可改用
自己的 OpenAI API Key。所有會議資料僅存在你的裝置本機。
