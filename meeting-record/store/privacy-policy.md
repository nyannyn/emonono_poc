# 隱私權政策 — Speech-to-Text：AI 語音轉錄

最後更新：2026-06-11

本政策說明「Speech-to-Text：AI 語音轉錄」（以下稱「本 App」）如何處理你的資料。
把本檔內容貼到任一可公開存取的網址（GitHub Pages / Notion 公開頁 / 你的網站），
該網址填入 App Store Connect 的「App 隱私權政策 URL」。

## 我們收集 / 處理的資料

- **音訊與逐字稿**：你錄製或上傳的會議音訊，以及轉錄後的逐字稿與會議記錄。
- **設定值**：你在 App 內輸入的偏好（語言、模型選擇；若使用「自己的 API Key」模式，
  你的 OpenAI API Key）。

## 資料儲存位置

- 音訊、逐字稿、會議記錄、成員與聲紋，**只存在你的裝置本機**（SQLite + 檔案系統）。
- 設定與 API Key 存在裝置的安全儲存區（iOS Keychain，透過 expo-secure-store）。
- 我們**沒有**帳號系統，**不會**把上述資料上傳到我們自己的伺服器保存。

## 第三方處理（語音轉文字與摘要）

為了把語音轉成文字並整理會議記錄，音訊與逐字稿會傳送給 **OpenAI** 處理：

- **內建模式（預設）**：透過本 App 營運者維護的代理伺服器轉送給 OpenAI；代理伺服器
  僅即時轉送，不長期保存你的音訊或逐字稿內容。
- **自己的 API Key 模式**：直接從你的裝置傳送給 OpenAI，使用你自己的 OpenAI 帳號。

OpenAI 對 API 資料的處理見其政策：<https://openai.com/policies/privacy-policy>。
依 OpenAI 現行 API 政策，透過 API 傳送的內容預設不用於訓練其模型。

## 麥克風

本 App 在你按下錄音時才使用麥克風，用途僅為錄製會議音訊以進行轉錄。

## 資料刪除

你可在 App 內刪除個別會議或清除所有會議資料。解除安裝 App 會移除所有本機資料。

## 兒童

本 App 不針對 13 歲以下兒童設計，也不會刻意收集其資料。
